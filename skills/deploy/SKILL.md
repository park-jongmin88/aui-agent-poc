# deploy - 엔드포인트 배포

## 절차 (POC 단계)
1. 배포 대상 모델명과 버전을 확인한다
2. **현재 POC 단계에서는 실제 배포 API가 연동되지 않았음을 안내한다**
3. 향후 흐름을 설명한다: 포털 REST API 호출 → 엔드포인트 생성 → URL 반환

## 주의
- 실제 API 호출을 시도하지 않는다
