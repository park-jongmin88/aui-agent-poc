# template_only

## 타입
**TEMPLATE** — 아무 파일도 없는 케이스 (처음부터 시작)

## 설명
데이터도, 모델도, 코드도 없는 완전히 빈 케이스입니다.
베이스 템플릿을 기반으로 처음부터 작성합니다.

## 파일 구성
- 없음 (source/ 폴더만 존재)

## 예상 init 모드
- **모드**: TEMPLATE
- **프레임워크**: sklearn (기본값)
- **자동 생성**: 기본 템플릿 복사, TODO 항목 6개

## 작업 가이드
1. `init` 실행 → 기본 템플릿 run.py 생성
2. 섹션 3~5를 직접 작성 (에이전트가 도움)
3. `validate` → 모든 TODO 해결 확인
4. `train`

## 적합한 사용자
- 처음부터 새 모델을 설계하는 경우
- 기존 코드/데이터 없이 시작하는 경우
